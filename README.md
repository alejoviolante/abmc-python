# Sistema ABMC en Python

Proyecto académico desarrollado en Python con interfaz gráfica utilizando Tkinter y base de datos SQLite.

## Tecnologías utilizadas
- Python
- Tkinter
- SQLite

## Funcionalidades
- Alta, baja, modificación y consulta de registros
- Registro de eventos mediante decoradores y patrón observador

## Ejecución
1. Clonar el repositorio
2. Ejecutar el archivo `controlador.py` desde el entorno de desarrollo

## Notas
La base de datos y los registros de eventos se crean en el directorio desde donde se ejecuta la aplicación.
Este proyecto no cuenta con instalador ni ejecutable.
